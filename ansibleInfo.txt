### Dossier important #1 ###
/etc/ansible/

contient :
  - Fichier de config d'Ansible : ansible.cfg
  - Fichier de config des hosts : hosts
  - dossier des roles : roles/



### lancement ###
ansible <host-pattern> [-m module_name] [-a args] [options]

### faire une execution du module command idempotente ###
avec une execution - command: /..../create_stuff.sh
--> utiliser l'attribut creates ou removes
creates : a filename or (since 2.0) glob pattern, when it already exists, this step will not be run.
removes : a filename or (since 2.0) glob pattern, when it does not exist, this step will not be run.
unarchive (decompression) aussi a un attribut creates.

--> il est possible de créer une "custom" idempotence
